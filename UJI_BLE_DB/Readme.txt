BLE measurements database for zones in the Library and Geteoc Lab from UJI and supporting material.
Version 1.0 05-12-2018

Directory content:

./      ->      Matlab scripts for usage examples.

data/   ->      Collection data. Within it, subfolder containing BLE RSS measurements and their associate data, 
		BLE beacon deployments, and shelves and pillars coordinates. For BLE RSS measurements, the measurements corresponding to 
		each zones are represented by four files: the RSS, the time, the coordinates, and the identifiers files.
		(Сбор данных. Внутри нее подпапка, содержащая измерения BLE RSS и связанные с ними данные, развертывания 
		маяков BLE, а также координаты полок и опор. Для измерений BLE RSS измерения, соответствующие каждой зоне, 
		представлены четырьмя файлами: RSS, время, координаты и файлы идентификаторов.)

load/   ->	Matlab functions for loading and filtering the measurements.

ips/	->      Matlab functions provinding sample IPS methods and some related utilities.

utils/	->	Utility function common to sample usage scripts.
